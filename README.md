## 📊 Projektüberblick

Analyse eines realen E-Commerce-Datensatzes zur Identifikation von Umsatztrends, Kundenverhalten sowie Produkt- und Verkäuferperformance.

Ziel war es, Rohdaten in eine strukturierte, analysierbare Datenbasis zu überführen und daraus verwertbare Business-Insights abzuleiten.

---

## 🔥 Zentrale Erkenntnisse

- Analyse der Umsatzentwicklung über Zeit
- Segmentierung von Kunden nach Kaufverhalten
- Identifikation schwacher Produktkategorien
- Vergleich von Verkäufern anhand ihrer Performance

---

## ⚙️ Technische Umsetzung

- Datenbereinigung und Transformation mit SQL
- Erstellung einer zentralen Analyse-View: `v_bestellungen`
- Einsatz von JOINs, Aggregationen und Window Functions
- Performance-Optimierung durch Indizes
- Vorbereitung der Datenstruktur für BI-Tools wie Power BI oder Tableau
